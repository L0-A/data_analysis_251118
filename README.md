# ☀️ [프로젝트 제목: 예시: Streamlit 기반 기상 관측 데이터 시각화 및 분석]

### 👨‍💻 1. 프로젝트 개요, 주제 및 선정 이유

#### 1.1 주제
파이썬 **Streamlit**을 활용하여 **과거 기상 관측 데이터**를 인터랙티브하게 분석하고 시각화하는 웹 애플리케이션 개발.

#### 1.2 주제 선택 이유
이전에 **Streamlit을 이용한 날씨 통계 프로젝트**를 진행한 경험을 바탕으로, 데이터 처리 능력과 웹 앱 구현 능력을 결합하여 **데이터 분석 결과를 사용자 친화적인 형태로 제공**하는 실무 능력을 보여주기 위해 선택했습니다. 또한, 공공 데이터를 다루고 시계열 데이터를 처리하는 경험을 쌓고자 했습니다.

### 📊 3. 데이터 분석 내용

본 프로젝트는 기상청 ASOS 일별 관측 데이터 (`OBS_ASOS_DD.csv`)를 활용하여 사용자가 원하는 조건에 따른 다양한 통계 분석 결과를 제공합니다.

#### 3.1 데이터 분석 과정
* **데이터 로드 및 정제:** `OBS_ASOS_DD.csv` 파일을 로드하고, `일강수량(mm)`의 결측치(NULL)를 0으로 대체 처리했습니다.
* **특징 공학 (Feature Engineering):** `일시` 컬럼을 `pd.to_datetime`으로 변환하여 `년`, `월`, `일` 컬럼을 추출해 필터링 기능을 구현하는 데 활용했습니다.

#### 3.2 주요 기능 (데이터 분석 내용)
1.  **홈 (`home`):** 프로젝트 소개 및 데이터 출처(`기상자료개방포털`, `국토지리정보원`) 정보 제공.
2.  **지점별 분석 (`show_spot`):**
    * 사용자 선택 (년/월, 계산할 값, 계산식, 정렬)에 따라 전국 **지점명**을 그룹핑하여 통계값을 계산합니다.
    * `Plotly Express`를 사용한 **막대 그래프**를 통해 지점별 통계값(평균 기온, 강수량 합 등)을 인터랙티브하게 비교합니다.
3.  **자료실 (`show_data`):**
    * **지점명, 년, 월, 일**을 기준으로 원본 데이터를 필터링하고, 특정 지표(기온, 강수량)를 기준으로 **정렬**하여 원본 데이터 조회 기능을 제공합니다.
