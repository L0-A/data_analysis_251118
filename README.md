# ☀️ [과제 제목: Streamlit을 활용한 기상 통계 분석]

### 👨‍💻 개요 및 목표

본 프로젝트는 파이썬의 **Streamlit**을 활용하여 **과거 기상 관측 데이터**를 분석하고, 사용자가 선택한 조건에 따라 통계 및 시각화 결과를 실시간으로 제공하는 웹 애플리케이션입니다.

* **핵심 목표:** 사용자가 원하는 기간 및 지표를 선택하여 **실시간으로 통계 및 시각화** 결과를 확인할 수 있도록 구현.
* **주요 기능:**
    1.  **지점별 분석:** 전국 지점별 기온, 강수량 등 통계값 계산 및 **막대 그래프** 시각화.
    2.  **자료실:** 사용자가 선택한 지점명, 년, 월, 일 조건에 따른 원본 데이터 **필터링 및 정렬** 기능 제공.
* **데이터셋:** 기상청 ASOS 일별 관측 데이터 (`OBS_ASOS_DD.csv`).

프로젝트 폴더가 다음과 같이 구성되어 있는지 확인합니다.
